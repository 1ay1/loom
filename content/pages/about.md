# About me

I'm a software engineer who enjoys building things from scratch to understand how they work.

This blog is built with **Loom**, a custom C++20 blog engine with a hand-rolled HTTP server. No frameworks, no dependencies, just the standard library and Linux syscalls.

## Interests

- Systems programming (C++, Rust)
- Networking and protocol design
- Developer tools
- Understanding things deeply
